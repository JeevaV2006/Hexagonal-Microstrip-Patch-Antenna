Hexagonal Microstrip Patch Antenna

This project presents the design of a hexagonal microstrip patch antenna loaded with a Split Ring Resonator (SRR) operating at 12.5 GHz (Ku-band). The antenna is modeled and simulated using ANSYS HFSS.
The design achieves a return loss of −19.5 dB, VSWR ≈ 1.2, and a gain of about 2.18 dB.
The SRR structure improves impedance matching and enhances antenna performance.
The antenna exhibits a stable broadside radiation pattern, making it suitable for satellite communication.
This design is compact, cost-effective, and easy to fabricate using FR-4 substrate.
Applications include DTH broadcasting, VSAT systems, and radar communication.
